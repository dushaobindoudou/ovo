## 1. 概述

本文档描述 Nudge 桌面助手的控制台窗口设计，提供调试、配置、状态监控等功能入口。

#### 配置项详情

| 配置项 | 类型 | 选项/默认值 | 说明 |
|--------|------|-------------|------|
| 监控间隔 | 下拉 | 1s, 3s, 5s, 10s, 30s | 5s |
| 监控目标 | 下拉 | 活动窗口, 指定窗口, 全屏 | 活动窗口 |
| 指定窗口 | 多选 | 窗口列表 | 仅当监控目标=指定窗口时显示 |
| OCR 引擎 | 下拉 | tesseract | 未来可扩展 |
| CLI 状态 | 只读 | - | 显示安装状态和版本 |
| 超时时间 | 下拉 | 10s, 30s, 60s, 120s | 30s |
| 模型 | 下拉 | sonnet, haiku | sonnet |
| 日志级别 | 下拉 | debug, info, warn, error | info |
| 语音输入 | 开关 | on/off | on |
| 语音输出 | 开关 | on/off | on |
| 唤醒键 | 输入 | 快捷键 | Cmd+Shift+Space |
| TTS 声音 | 下拉 | 晓晓, 云希, 云扬, 小艺 | 晓晓 |
| 自动路由 | 开关 | on/off | on |
| 默认人格 | 下拉 | work, friend, learning, creative, focus, auto | auto |
| 应用规则 | 列表 | App→Personality | 可添加/删除 |
| 自我迭代 | 开关 | on/off | on |

#### Agent 定义

| Agent | 描述 | 输入 | 输出 |
|-------|------|------|------|
| 场景判断 | 判断当前场景和 Soul | OCR文本 + 窗口信息 | scene, confidence, suggestedSoul |

| 执行器 | 执行具体操作 | Action 参数 | 执行结果 |

## 5. 技术实现

### 5.1 文件结构

```
src/
├── components/
│   ├── Console/
│   │   ├── ConsoleLayout.tsx      # 主布局（导航+内容）
│   │   ├── ConsoleNav.tsx         # 左侧导航栏
│   │   ├── DebugPanel.tsx        # 调试页面
│   │   ├── SettingsPanel.tsx    # 设置页面
│   │   ├── AgentPanel.tsx        # Agent 页面
│   │   ├── StatusPanel.tsx       # 状态页面
│   │   └── AboutPanel.tsx        # 关于页面
│   │   └── WindowList.tsx        # 窗口列表组件
│   │   └── OCRResult.tsx         # OCR 结果组件
│   │   └── LogViewer.tsx         # 日志查看器
```

### 5.2 路由设计

控制台使用 hash 路由：
- `#console` - 主控制台入口（显示调试页）
- `#console/debug` - 调试页
- `#console/settings` - 设置页
- `#console/agent` - Agent 页
- `#console/status` - 状态页
- `#console/about` - 关于页

### 5.3 IPC 接口

需要添加的 IPC：

| IPC | 方向 | 参数 | 返回 |
|-----|------|------|------|
| `windows:list` | main→renderer | - | WindowInfo[] |
| `windows:active` | main→renderer | - | WindowInfo |
| `monitor:start` | renderer→main | interval, target | void |
| `monitor:stop` | renderer→main | - | void |
| `monitor:result` | main→renderer | OCRResult | void |
